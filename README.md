# ESP32 IoT Weather Monitoring System

## рџ“Њ Overview

The ESP32 IoT Weather Monitoring System is a smart environmental monitoring project that utilizes an ESP32 microcontroller and a DHT11 sensor to collect real-time temperature and humidity data. The collected data is transmitted over Wi-Fi and displayed on Blynk mobile and web dashboards, enabling users to monitor environmental conditions remotely from anywhere.

## вљ™пёЏ Working Principle

1. The DHT11 sensor continuously measures temperature and humidity.
2. The ESP32 reads sensor values through its GPIO pins.
3. The collected data is processed by the ESP32.
4. Using Wi-Fi connectivity, the ESP32 transmits sensor data to the Blynk Cloud.
5. The Blynk platform updates the mobile and web dashboards in real time.
6. Users can remotely monitor environmental conditions from anywhere with internet access.
